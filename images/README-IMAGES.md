# Image Requirements for place.html

You need to add the following WebP images to this folder:

## Required Images:

1. **new-zealand-small.webp**
   - For mobile view (viewport < 700px)
   - Recommended dimensions: 600px wide
   - Subject: New Zealand landscape (mountains, lakes, etc.)
   - Format: WebP

2. **new-zealand-large.webp**
   - For desktop view (viewport >= 700px)
   - Recommended dimensions: 1200px wide
   - Subject: New Zealand landscape (mountains, lakes, etc.)
   - Format: WebP

## Where to Get Images:

- **Pexels**: https://www.pexels.com/
- **Unsplash**: https://unsplash.com/

Search for "New Zealand landscape" or similar terms.

## Converting to WebP:

You can convert images to WebP format using:
- Online tools: https://cloudconvert.com/jpg-to-webp
- Photoshop: Save As > WebP
- Command line tools (cwebp)
